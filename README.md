# pos_kiosk

Inventory & POS Kiosk

A Flutter mobile application for managing inventory.

Features:
- Create products
- View inventory
- Edit products
- Delete products
- Dialog-based forms
- Swipe-to-delete functionality

Built with:
- Flutter
- Dart
